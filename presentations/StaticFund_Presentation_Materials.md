# 📋 PART 3: Business Model Canvases

---

## Business Model Canvas #1: Commission Model (PRIMARY)

| Block | Content |
|-------|---------|
| **Customer Segments** | 1) Households wanting to reduce electricity costs 2) Solar installers seeking qualified leads |
| **Value Propositions** | AI appliance scanning, Accurate energy audits, Tailored solar sizing, Energy-saving tips, Installer marketplace |
| **Channels** | Mobile app (iOS/Android), Website, Social media, Solar company partnerships |
| **Customer Relationships** | Self-service app, Automated notifications, Personal support, Solar repair connections |
| **Revenue Streams** | Commission per lead/customer sent to installers |
| **Key Resources** | Mobile app & website, AI model, Own servers, Appliance database, Installer network, Technical team |
| **Key Activities** | Onboard installers, Acquire users, Match leads, Platform development, Customer support, Marketing |
| **Key Partners** | Solar installation companies |
| **Cost Structure** | App development, Server hosting (electricity), AI API costs, Advertising |

---

## Business Model Canvas #2: Subscription Model

| Block | Content |
|-------|---------|
| **Customer Segments** | Same as Canvas #1 |
| **Value Propositions** | Same as Canvas #1 |
| **Channels** | Same as Canvas #1 |
| **Customer Relationships** | Same as Canvas #1 |
| **Revenue Streams** | **Monthly subscription for installers** (e.g., R500-R2,000/month to be listed and receive leads) |
| **Key Resources** | Same as Canvas #1 |
| **Key Activities** | Same as Canvas #1 |
| **Key Partners** | Same as Canvas #1 |
| **Cost Structure** | Same as Canvas #1 |

**Pros:** Predictable recurring revenue, easier financial forecasting  
**Cons:** Higher barrier for installer onboarding, need to prove value upfront

---

## Business Model Canvas #3: Transaction Fee Model

| Block | Content |
|-------|---------|
| **Customer Segments** | Same as Canvas #1 |
| **Value Propositions** | Same as Canvas #1 |
| **Channels** | Same as Canvas #1 |
| **Customer Relationships** | Same as Canvas #1 |
| **Revenue Streams** | **Percentage of installation sale** (e.g., 2-5% of contract value) |
| **Key Resources** | Same as Canvas #1 |
| **Key Activities** | Same as Canvas #1, plus sales tracking/verification |
| **Key Partners** | Same as Canvas #1 |
| **Cost Structure** | Same as Canvas #1 |

**Pros:** Aligned with installer success, higher revenue per transaction  
**Cons:** Requires tracking actual sales, longer payment cycles, more complex
